# Blog sur de la rédaction d'histoire

Bonjour à tous, ce répertoire github va consister en une sorte de réflection / travail personnel sur tout ce qui est création de monde / création d'histoires.

Mon objectif va être de lister / rassembler / synthétiser / modéliser le plus de concepts possibles afin de faire une sorte de base de donnée / guide / banque d'idée pour moi même et avec un peu de chance ca pourra aussi être utile pour d'autres personnes.

Je vais essayer d'organiser tout cela de la façon la plus modulaire possible. Donc pour chaque partie, il y aura une sorte d'index / page d'acceuil qui va présenter et apporter des éléments généraux, avant de rediriger vers d'autre fichiers plus spécifiques qui pourront aussi rediriger vers d'autres fichiers.

Pour l'instant, je vais tout rédiger en français car c'est un peu plus rapide pour moi de rédiger en français qu'en anglais, mais si j'arrive à faire quelque chose de bonne qualité, je travaillerais aussi sur une traduction en anglais.
